Erlang-measurements

Measurement converter for erlang

Download this project as a .zip file Download this project as a tar.gz file

Welcome to Erlang Measurements.

This application contains useful functions to convert values from one measurement/metric to another.
Available conversions are currency, metric/imperial and temperature.
The application uses a table to convert where applicable and if you add more measurements to the table please send me a pull request.
